# **App Name**: SimpliPlan - Celebrating People

## Core Features:

- Dynamic Budget Input: Input fields for each item to adjust metric, quantities and unit prices, automatically calculating totals. The input must preserve its type in any exports.
- Category Management: Enable users to collapse and expand budget categories for easier navigation. Also must have a category summary pill.
- Real-Time Calculation: Automatically recalculate totals for each item and category, as well as the grand total, when inputs are changed.
- Summary View: A clear summary table displaying totals for each category, mirroring the totals shown in each respective pill, and the overall grand total. Must expand the category to its sub-category with sub-totals.
- Comment field: Include a multi-line textfield, also called a text area, so that the user can add some context on how each cost will be determined.

## Style Guidelines:

- Primary color: Teal (#00A693), drawn from the 'SimpliPlan' logo, offering a sense of modernity and sophistication.
- Secondary color: Dark Gray (#4A4A4A), for the text 'Celebrating People', providing a grounded and reliable feel.
- Background color: Light Gray (#F0F4F7), providing a neutral and clean backdrop to enhance readability.
- Body and headline font: 'PT Sans', a humanist sans-serif to combine modernity with a touch of warmth, ensuring readability and approachability.
- Use simple, clear icons for budget categories to aid navigation. These should be consistently styled and minimally detailed.
- The layout should be clean and intuitive, with clear separation between categories and items. Utilize whitespace effectively to prevent the interface from feeling cluttered.
- Subtle transitions when categories are expanded or collapsed, providing gentle feedback without distracting the user.
